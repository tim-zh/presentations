AdaBoost

yi from {-1, 1}. for each model M

  • M.train(X, Y, W)
  • Y' = M.predict(X)
  • e = ∑ wi * e-yi * y'i
  • a = 1/2 * log (1 - e)/e
  • wi *= exp(-a * yi * yi')
  • W /= ∑ wi
